## Interactive Components

Here's a simple counter component:

```jsx
import { useState } from 'react'

export const Counter = () => {
  const [count, setCount] = useState(0)
  return (
    <div className="flex gap-4 items-center p-4 bg-gray-100 rounded-lg">
      <button 
        onClick={() => setCount(c => c - 1)}
        className="px-4 py-2 bg-red-500 text-white rounded"
      >
        -
      </button>
      <span className="text-xl font-bold">{count}</span>
      <button 
        onClick={() => setCount(c => c + 1)}
        className="px-4 py-2 bg-green-500 text-white rounded"
      >
        +
      </button>
    </div>
  )
}
```

## Quick Start Guide

1. Install dependencies:
   ```bash
   npm install next mdx @mdx-js/react
   ```

2. Create your first MDX file:
   ```mdx
   # Hello MDX
   This is your first MDX file!
   ```

3. Import and use in your React components:
   ```jsx
   import MyMDXContent from './content.mdx'
   ```
